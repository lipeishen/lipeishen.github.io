关于泛型的问题，主要是解决集合中的对象，取出时变为Object， 还需强制转化为原类型，解决办法就是使用泛型<>

list<String> list= ArrayList<String>();

Java 中集合主要分为两类接口 Collection 和Map, Collection又分为Set、Queue、和List三类接口，其中Set 是无序的不能有重复的元素，List 可以。

泛型中List<Collection> 并不是List<set>的父类，这一点一定要搞明白。一般使用通配符“？”List<?>,它可以是任意List<>的父类，要想限制子类，需要
List<? extends Collection>
